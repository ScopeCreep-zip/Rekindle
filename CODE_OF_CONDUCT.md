# Code of Conduct

The Rekindle project adopts the **Contributor Covenant, version 2.1**, as
its code of conduct for all project spaces — the GitHub repository, issue
tracker, pull request discussions, security advisories, and any chat
channels operated by the project.

The canonical text is published at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

We follow it as written, with the project-specific enforcement details
below. By participating in this project you agree to abide by the linked
Covenant.

## Scope

This code of conduct applies to all project spaces, and also applies when
an individual is officially representing the project in public spaces
(for example, posting via an official social account or acting as a
designated representative at an online event).

## Enforcement

Concerns about behavior in any project space can be reported confidentially
by opening a private message to the maintainers via:

- **Email:** `conduct@scopecreep.zip`
- **GitHub:** open a private vulnerability report at
  <https://github.com/ScopeCreep-zip/Rekindle/security/advisories/new>
  (this private channel can be used for conduct concerns as well as
  security reports — please mark the report `[CONDUCT]` in the title).

All reports will be reviewed and investigated promptly and fairly. The
project maintainers are obligated to respect the privacy and security of
the reporter of any incident.

## Enforcement guidelines

Project maintainers will follow the Community Impact Guidelines published
alongside the Contributor Covenant 2.1 when determining the consequences
of any action they deem in violation of this Code of Conduct. The four
escalation levels (Correction → Warning → Temporary Ban → Permanent Ban)
are defined in the linked canonical text.
